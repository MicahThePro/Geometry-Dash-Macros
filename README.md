# Geometry Dash Macro Collection

A collection of macros I’ve created for showcasing Geometry Dash levels.  
These macros are intended for **content creation, level previews, and demonstrations**.

## 📁 Repository Structure
- `macros/` — All macro files, organized by difficulty and category  
- `previews/` — Optional preview images or GIFs of the runs  
- `metadata/` — Extra info about each macro (FPS, version, notes)

## 🎮 What’s Included
- Clean, synced macros for level showcases  
- Multiple FPS versions when needed  
- Notes on timing, sync, or special behavior  
- Practice-route macros for harder levels

## ⚙️ How to Use
1. Download the macro file you want  
2. Import it into your preferred GD macro tool  
3. Make sure your FPS matches the macro’s FPS  
4. Play or record the showcase

## 🧾 Macro Metadata Format
Each macro may include a `.json` file with:
```json
{
  "level": "Level Name",
  "creator": "Creator Name",
  "fps": 240,
  "length": "1m 12s",
  "notes": "Recorded on 2.2081"
}
